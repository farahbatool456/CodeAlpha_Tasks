# Screenshots Folder

Place your app screenshots here after running the project.

## Suggested Screenshots to Capture

1. `light_mode.png`     — Full app in light mode with a sample conversation
2. `dark_mode.png`      — Full app in dark mode
3. `sidebar.png`        — Sidebar with stats and category filter open
4. `confidence.png`     — A response showing High confidence badge
5. `no_match.png`       — The fallback response when no FAQ matches
6. `welcome.png`        — The welcome screen before any chat starts
7. `mobile.png`         — App on a narrow browser window (responsive view)

## How to Take Screenshots

### Windows
- Press `Windows + Shift + S` to open the snipping tool
- Select the region you want to capture
- Paste into Paint or any image editor and save as PNG

### macOS
- Press `Cmd + Shift + 4` then drag to select a region
- Screenshot saves automatically to your Desktop

### Browser Screenshot (Any OS)
- Open Chrome DevTools with `F12`
- Press `Ctrl + Shift + P` (Windows) or `Cmd + Shift + P` (Mac)
- Type "screenshot" and select "Capture full size screenshot"

## Tips for a Good Portfolio Screenshot
- Use a clean browser window with no extra tabs visible
- Resize the browser to show the full chat interface
- Use a real conversation (not just the welcome screen)
- Include both light and dark mode versions
