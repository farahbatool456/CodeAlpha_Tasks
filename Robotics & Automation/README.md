## ⚙️ Task 2 — Automation System Design: Automated Packaging System

**File:** `Task2_Automated_Packaging_System.pdf`

A complete engineering design document for a sensor-integrated automated packaging system controlled by a PLC.

### System Specs
| Parameter | Value |
|-----------|-------|
| Throughput | 1,200 units/hour |
| Defect Rate | 0.9% |
| Controller | Siemens S7-1500 PLC |
| Communication | Profibus DP / EtherNet IP |
| Safety Standard | IEC 62061 SIL 2 / ISO 13849 PLd |
| Payback Period | 2.4 years |
| OEE | 86.5% |

### Includes
- 7-station process flow diagram
- Full sensor selection table (vision, weight, proximity, temperature)
- PLC control logic table
- KPI performance analysis
- ROI & cost comparison table
- Safety system design
